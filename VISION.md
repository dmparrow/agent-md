# Agent-MD Vision

Agent-MD is a Git-native software delivery harness that converts backlog items into auditable software changes through context packs, patch promotion, validation gates, and handover artifacts.

## Principles

- Agent proposes diffs.
- Harness promotes diffs.
- Git records truth.
- Tests prove completion.
- Handover preserves context.

The harness owns workflow. Agents are replaceable.
